### Project: CLI-Tacts

CLI-Tacts is a desktop application for CS2040S Teaching Assistants to manage students, tutorial groups, and attendance efficiently through a CLI, with a JavaFX GUI.

Given below are my contributions to the project:

1. Tutorial group field
* Added tutorial group support as a dedicated student field.
* Enforced single tutorial group per student and integrated it across model, parser, command, storage, and UI layers.

2. Input restrictions and regex validation
* Added and refined validation checks for command arguments (e.g., student ID, email, phone number, tutorial group).
* Strengthened parser-side validation and exception handling for invalid inputs.

3. Documentation updates
* Updated User Guide, Developer Guide, and index documentation to reflect CLI-Tacts features and workflows.
* Revised command formats and examples to match the implemented behavior.
